# jTOPMODEL

TOPMODEL (Beven & Kirkby 1979) extracted from [SYMFLUENCE](https://github.com/DarriEy/SYMFLUENCE).

Dual-backend (JAX + NumPy) implementation of TOPMODEL with degree-day snow module,
saturation-excess overland flow, and channel routing.

## Installation

```bash
pip install jtopmodel
```
